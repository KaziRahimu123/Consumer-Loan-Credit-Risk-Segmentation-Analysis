# Description
This project analyzes loan and credit application data to identify high-quality, low-risk customer segments for consumer lending. Using income, employment stability, demographics, and household burden, it delivers data-driven insights to improve loan approvals, pricing strategies, and targeted marketing without increasing portfolio risk.
